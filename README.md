# ⚡ Calculadora da Lei de Ohm (Kotlin)

Aplicação desenvolvida em Kotlin para calcular valores elétricos com base na Lei de Ohm, permitindo determinar tensão, corrente e resistência de forma simples e rápida.

# 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de aplicar conceitos de programação utilizando Kotlin, aliado a fundamentos básicos da eletricidade. <br>
<br>
A calculadora utiliza a fórmula da Lei de Ohm para resolver problemas envolvendo: <br>

Tensão (V) <br>
Corrente (I) <br>
Resistência (R)

# ⚡ Lei de Ohm

A relação entre as grandezas elétricas é dada por: <br>

V = R × I <br>
I = V / R <br>
R = V / I

# 🚀 Funcionalidades

✔ Cálculo de tensão (V) <br>
✔ Cálculo de corrente (I) <br>
✔ Cálculo de resistência (R) <br>
✔ Interface simples e direta (dependendo da implementação)
